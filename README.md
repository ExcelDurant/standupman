# StandUpMan

**StandUpMan** is an agile tool used to manage scheduled standup-notes.


## Installation 

- `git clone https://github.com/Fenn-CS/standupman`
- `cd standupman`
- `cp .env.example .env`
- `npm install`
- `npm run dev`
